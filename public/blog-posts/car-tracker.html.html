System Overview

The vehicle tracker collects location, movement, and sensor data from the vehicle and transmits it over GSM to a cloud server. The MT2503 SoC manages GNSS positioning and GSM connectivity, while onboard accelerometers and gyroscopes detect harsh driving events. The firmware fuses these inputs, formats them into efficient telemetry packets, and pushes them to the backend for analysis.

System Architecture

The architecture combines a low-power MT2503 GSM-GNSS core with sensor interfaces and optional expansion modules. The SoC communicates with the accelerometer and gyroscope over I2C/SPI, while GSM and GNSS run in parallel for continuous tracking. The system integrates with cloud APIs via GPRS for data reporting.

Hardware Components

1 – Core Module

  • SoC: MediaTek MT2503 (GSM + GNSS)
  • Connectivity: GSM 850/900/1800/1900 MHz, GPS/GLONASS/Beidou
  • Interfaces: UART, I2C, SPI, GPIO

2 – Sensors

  • 3-axis accelerometer
  • 3-axis gyroscope
  • Optional temperature and fuel level sensors

3 – Other GSM-GNSS Modules Tested

  • Quectel
  • Simcom
  • AiThinker

Software Components

  • Firmware: Based on vendor code from new-mobie, modified to support additional sensors and data formats
  • Protocols: Custom binary telemetry over TCP/UDP, SMS fallback
  • GNSS Processing: NMEA parsing, assisted-GPS support

Frontend Application

Operators accessed tracking data via a web-based dashboard. The interface displayed real-time positions, historical trip data, and alerts for harsh driving behavior.

System Monitoring

  • Signal strength and GPS lock status
  • Sensor calibration state
  • Battery voltage and device uptime

Performance Highlights

  • Reliable GSM and GNSS connectivity in urban and rural areas
  • Accurate detection of braking and acceleration events
  • Low power consumption, enabling multi-day operation on vehicle battery backup

Conclusion

The Multifunctional Vehicle Tracker project demonstrated the feasibility of delivering advanced telematics features in a compact and cost-efficient form factor. Overcoming challenges of limited documentation and vendor support, the team successfully integrated GSM, GNSS, and motion sensors into a robust solution ready for deployment in fleet and consumer markets.
