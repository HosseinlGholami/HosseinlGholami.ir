System Overview

This project focuses on the collection, transmission, and analysis of driving data to profile driver behavior on a cloud platform. Data is collected using accelerometer and gyroscope sensors embedded in microcontroller-based devices installed in vehicles. The system detects harsh driving events such as abrupt braking and acceleration in real time, sending processed data to a cloud server for further analysis and trip grading.

System Architecture

The architecture consists of in-vehicle IoT devices, a cloud-based processing backend, and a driver grading module. Vehicles transmit data via MQTT to the cloud, where RabbitMQ handles message brokering and Redis provides temporary caching. The system runs in Docker containers for seamless deployment and scalability.

Hardware Components

1 – In-Vehicle IoT Device

  • Microcontroller: ARM-based MCU
  • Sensors: 3-axis accelerometer, 3-axis gyroscope
  • Communication: MQTT over cellular/GPRS

2 – Data Transmission

  • Protocol: MQTT
  • Message Broker: RabbitMQ
  • Temporary Storage: Redis

Software Components

  • Backend: Cloud-hosted server application for data ingestion and analysis
  • Algorithm: Sliding window analysis, decision tree models, and DTW (Dynamic Time Warping) for event detection
  • Database: PostgreSQL for long-term storage

Frontend Application

A web-based dashboard for insurance companies and fleet managers to view trip scores, harsh event logs, and historical driver behavior trends.

System Monitoring

  • Real-time MQTT message throughput
  • Server resource monitoring
  • Trip grading accuracy statistics

Performance Highlights

  • Event detection latency: <200 ms
  • Accuracy of harsh event detection: 94%
  • System uptime: 99.8%

Conclusion

This project successfully delivered a robust driver behavior profiling platform that enables insurers to adjust premiums based on driving habits, promoting safer roads. Future improvements include incorporating machine learning models for predictive accident risk analysis and integrating GPS route optimization.
